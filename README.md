# Clínica Patitas Felices

Este proyecto es una página web sencilla para la clínica veterinaria **Patitas Felices**.  
Incluye varias secciones: inicio, galería, presupuesto, contacto y aviso legal.

---

## Estructura del proyecto

- index.html → Página principal  
- VIEW/  
  - galeria.html → Galería de imágenes  
  - presupuesto.html → Formulario de presupuesto  
  - contacto.html → Información de contacto y mapa  
  - aviso-legal.html → Aviso legal  
- css/  
  - estilos.css → Estilos generales  
- js/  
  - noticias.js → Noticias dinámicas  
  - galeria.js → Funciones de la galería  
  - presupuesto.js → Cálculo del presupuesto  
  - validaciones.js → Validaciones de formularios  
- logo-clinica1.png → Logo de la clínica  
- favicon.ico → Icono del sitio  
- README.md → Archivo de documentación  

---

## Tecnologías usadas

- HTML5 para la estructura  
- CSS3 para los estilos  
- JavaScript para la parte dinámica  
- Leaflet para el mapa interactivo  
- Font Awesome para los iconos  

---

## Cómo usarlo

1. Descargar el proyecto.  
2. Abrir `index.html` en el navegador.  
3. Navegar por las distintas páginas desde el menú.  

---

## Comentarios

- Se ha intentado que el código sea válido según W3C.  
- Los formularios tienen validaciones básicas.  
- El mapa de contacto usa la ubicación del usuario si da permiso.  
- El pie de página es igual en todas las páginas para mantener uniformidad.
